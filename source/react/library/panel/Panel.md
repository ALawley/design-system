Secondary `Panel`:

```
<Panel secondary>
  I am a happy secondary panel.
</Panel>
```

Removable `Panel`:

```
<Panel onRemove={ () => alert('Panel removed!') }>
  I am the Frito Bandito.
</Panel>
```

`Panel` with `Content`:

```
<Panel>
  <Content>
    <h2>Important!</h2>
    <p>I am the first paragraph.</p>
    <p>I am a Silkie chicken.</p>
  </Content>
</Panel>
```
